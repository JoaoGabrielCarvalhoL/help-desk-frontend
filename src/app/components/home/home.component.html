Help Desk

Olá! Seja bem-vindo ao sistema Help Desk!
Esse é um sistema para gerenciamento de chamados. Esse sistema foi desenvolvido em Angular e Spring Boot 2.
O projeto tem como objetivo abordar os conceitos de aplicações SPAs, Autenticação e Autorização Tokens JWT e muito mais. O código fonte do projeto pode ser acessado clicando no link Github no menu.
